# CrowdZK: Privacy-Preserving Crowdsourcing using Zero-Knowledge Proofs

## Overview
This repository provides the implementation and experimental evaluation of CrowdZK, a privacy-preserving decentralized crowdsourcing framework using zk-SNARKs (Groth16). The system prevents solution probing attacks by eliminating observable feedback signals.

## Components
- Smart Contract (Solidity verifier)
- ZK Circuits (Circom)
- Proof Generation (snarkjs)
- Attack Simulation (Python)

## Environment
- Python 3.11
- Node.js (for snarkjs)
- Circom 2.x
- Solidity (Remix IDE)
